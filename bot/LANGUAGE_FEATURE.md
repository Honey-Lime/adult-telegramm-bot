# Функционал выбора языка / Language Selection Feature

## Описание

Полная поддержка двух языков (Русский/English) с автоматическим определением и сохранением в базе данных.

## Изменения

### 1. База данных (`database.py`)

- Добавлен столбец `language` в таблицу `users` (по умолчанию 'ru')
- Функция `get_user_language(user_id)` - получение языка пользователя
- Функция `set_user_language(user_id, language)` - установка языка пользователя
- Обновлена функция `get_or_create_user()` - добавлен параметр `language`
- Обновлена функция `update_user_profile()` - добавлен параметр `language_code`
- **Автоматическая миграция**: `init_db()` проверяет и добавляет столбец `language` при запуске

### 2. Локализация (`locales.py`)

- Все тексты бота разделены на два языка
- Функция `get_text(lang, key, **kwargs)` - получение текста по ключу
- Функция `get_language_name(lang)` - название языка
- Поддержка форматирования строк через kwargs

### 3. Клавиатуры (`keyboards.py`)

- Все функции клавиатур принимают параметр `lang`
- Кнопки отображаются на выбранном пользователем языке

### 4. Бот (`bot.py`)

- Метод `get_user_lang(chat_id)` - получение языка пользователя
- Обновлён `cmd_start()` - определение языка из `message.from_user.language_code`
- Обновлены `_update_user_profile_from_message()` и `_update_user_profile_from_callback()` - сохранение языка
- Обновлены все обработчики команд и колбэков с использованием локализации
- Обновлены методы отправки сообщений (`send_menu`, `send_picture`, etc.)

## Как это работает

### 1. При первой регистрации:

- Бот определяет язык из `message.from_user.language_code`
- Если язык начинается с 'ru' → устанавливается 'ru', иначе 'en'
- Язык сохраняется в базе данных

### 2. При каждом взаимодействии:

- Язык пользователя обновляется из данных Telegram (автоматически)

### 3. Ручное изменение:

- Пользователь нажимает кнопку "🌐 Language / Язык" в главном меню
- Выбирает один из двух языков: 🇷🇺 Русский или 🇺🇸 English
- Выбранный язык сохраняется в базе данных

## Использование

### Для пользователей:

1. Нажмите кнопку "🌐 Language / Язык" в главном меню
2. Выберите нужный язык: 🇷🇺 Русский или 🇺🇸 English

### Для разработчиков:

```python
# Получить язык пользователя
lang = database.get_user_language(user_id)

# Установить язык
database.set_user_language(user_id, 'ru')  # или 'en'

# Получить текст
from locales import get_text
text = get_text(lang, 'main_menu_text')

# Получить текст с переменными
text = get_text(lang, 'donate_menu_text', coins=100)
```

## Добавление нового языка

1. Добавьте новый ключ в `LOCALES` в файле `locales.py`:

```python
'en': {
    'main_menu_text': '👋 Here you can choose...',
    'btn_anime': '🖼 Anime',
    # ... остальные переводы
}
```

2. Обновите `get_language_name()`:

```python
def get_language_name(lang: str) -> str:
    names = {
        'ru': 'Русский',
        'en': 'English',
        'fr': 'Français'  # новый язык
    }
    return names.get(lang, 'Русский')
```

## Миграция

Миграция выполняется **автоматически** при запуске бота. Функция `init_db()`:

1. Проверяет наличие столбца `language` в таблице `users`
2. Если столбец отсутствует → добавляет его со значением по умолчанию `'ru'`
3. Все существующие пользователи получают русский язык по умолчанию

Никаких дополнительных действий не требуется! 🚀
